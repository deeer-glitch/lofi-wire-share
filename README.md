# lofi-wire-share

오늘의집 PD 디어의 Lo-fi 와이어프레임 공유 레포.

## 주의

- 이 레포의 모든 와이어는 **자동 마스킹된 공유본**입니다.
- 카피·색·가격·로고는 **더미 데이터**이며, 실제 디자인·정책·내부 SSOT와 다를 수 있어요.
- 내부 시스템 정보(PRD·Jira·Slack·실측 데이터 등)는 마스킹되어 있습니다.
- 외부 공유 목적의 **참고용 시안**입니다.

## 구조

```
wires/
└── {project}/
    └── {topic}-{kind}{-v?}.html
```
